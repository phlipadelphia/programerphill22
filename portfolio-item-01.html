PasswordRegurgitator

A Secure Password Generator

With this project, it truly marked a turning point early into my education. It was then where I begun to fully grasp and appreciate front-end development and especially JavaScript.

This application is simply a secure password generator, one that could me implemented into many other sorts of applications. It starts with a series of user prompts, made to guide users to creating a specific password unique to them based on certain parameters to ensure security.

This project was made in Visual Studio Code with HTML, CSS and JavaScript.
