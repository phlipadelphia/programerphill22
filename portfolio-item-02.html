Where to go? Travel IO

Search a City for Weather, Events and Lodging!

Behold the labors of my first group collaborative effort, made by me and a handful of (now former) classmates. A simple travel application that prompts a user to search any given city for their respective in-the-moment details, such as weather, hotel vacancies, restuarants and events!

After a certain point in our education, we decided we wanted to flex the knowledge we had thus far and came up with a idea for a site that utilized what we knew. The idea came to us after thinking specifically about user intercontivity on a local level with any given city. More than anything this was a chance we gave ourselve to work on and demonstrate our working knowledge of Web API's as we applied various types of api's in order to retrieve that type of information local to any entered city.

This project was largely put together in visual studio code and github, using HTML, CSS, JavaScript and a handful of Web APIs. A large part of my contribution was on some of the WEB API and CSS for the landing page.
